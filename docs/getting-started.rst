Getting Started
===============

How to use this documentation site:

1. Add new mechanism writeups under ``mechanisms/``.
2. Add reusable software feature writeups under ``software/``.
3. Capture each season under ``season-archive/``.

Each page should include:

- What problem was being solved
- Design or implementation details
- What worked, what failed, and why
- Links to CAD, photos, and code references
